# 飞行模拟器 (Flight Simulator)

这是一个基于 Win32 API 和 GDI 的 C++ 飞行模拟程序。

## 功能特性
- 🛫 **物理模拟**：包含迎角、失速和重力下坠逻辑。
- ⚠️ **多重警报**：具备近地警告、地障（铁塔）警告和失速警告。
- 🎨 **双缓冲绘图**：解决画面闪烁问题，实现流畅渲染。
- 🔊 **声音反馈**：不同优先级的警报音和坠毁音效。

## 控制方式
- **方向键**：控制飞行姿态和油门。
- **R 键**：游戏重置。

## 编译运行
使用 Visual Studio 打开，确保配置了 Windows SDK。
